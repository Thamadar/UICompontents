Репозиторий с UI-компонентами. На данный момент это пока по сути некий графический редактор с элементарными геометрическими объектами. <br>
Клиенсткая часть реализована на двух платформах: WPF и Avalonia. <br>
ТЗ: <br>
"Необходимо привести пример программы на С#, которая выводит на экран различные геометрические фигуры: круг, эллипс, треугольник, квадрат, четырёхугольник и т.п. Глубина проработки примера (количество фигур, параметры, методы, дизайн) на Ваше усмотрение. В задании Вам необходимо продемонстрировать умение использовать ООП. UI-часть должна быть реализована с применением MVVM и AvaloniaUI." <br>
